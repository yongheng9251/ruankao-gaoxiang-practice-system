# 软考高项智能刷题系统

这是一个基于 Web 的软考高级信息系统项目管理师刷题系统，支持章节练习、随机练习、错题重练等功能。

## 功能特性

- 📚 支持多章节选择练习
- 🎯 顺序练习、随机练习、错题重练模式
- ⏱️ 计时考试模式
- 📊 实时统计学习进度
- 💾 进度保存与导入导出
- 📱 响应式设计，支持移动端
- 🔖 题目标记功能
- 🖱️ 键盘快捷键支持
- 📲 PWA 支持（可安装到桌面）

## 使用方法

1. 克隆项目到本地
2. 直接打开 `index.html` 即可使用
3. 点击"选择文件"按钮上传 `questions.json` 题库文件，或使用"加载示例数据"

## 键盘快捷键

- `←` / `→` : 上一题 / 下一题
- `A` / `B` / `C` / `D` : 选择选项
- `ESC` : 返回首页

## 技术特点

- 纯前端实现，无需服务器
- 本地存储用户进度
- 支持拖拽上传题库文件
- 渐变色彩设计，美观易用

## 数据格式

题库文件需为 JSON 格式，包含以下字段：
- `id`: 题目唯一ID
- `chapter`: 所属章节
- `question`: 问题内容
- `options`: 选项对象
- `answer`: 正确答案
- `analysis`: 答案解析
